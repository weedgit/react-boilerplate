# MERN Boilerplate

## Quick Start

#### Setup
```
brew install mongodb
npm install
```

#### for Development

Start the database
```
mongod
```

Start the client
```
npm run dev
```

Start the server
```
npm start
```

#### for Production

Start the database
```
mongod
```

Start the Server
```
npm run prod
```

#### Other Commands

```
npm run build
npm test
npm run test:verbose
npm run lint
npm run lint:fix
```
